# 🍕 FastAPI Food Delivery Backend

## 📌 Project Overview
This is a backend system built using FastAPI that simulates a real-world food delivery application.

## 🚀 Features
- RESTful APIs using FastAPI
- Pydantic data validation
- CRUD operations
- Cart → Checkout workflow
- Search, sorting, pagination
- Swagger API testing

## 🛠 Tech Stack
- Python
- FastAPI
- Uvicorn
